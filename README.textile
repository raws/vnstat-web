vnstat-web is a little tool I wrote to make a few simple graphs from "vnstat":http://humdi.net/vnstat/ network usage data. It is not the raddest thing ever, but it quite simply does what I would like it to do.

!vnstat-web/raw/master/images/readme-demo.png(Screenshot of vnstat-web)!

h3. Requirements

* "vnstat":http://humdi.net/vnstat/ >= 1.10
* PHP >= 5.2.0
